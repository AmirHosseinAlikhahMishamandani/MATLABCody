Problem 1. Times 2 - START HERE

Try out this test problem first.
Given the variable x as your input, multiply it by two and put the result in y.
Examples:
% Input x = 2
% Output y is 4
% Input x = 17
% Output y is 34

Scratch Pad

x = 2
x = 2
 
fprintf('The result of multiplying %.2f by 2 is equal to %.2f!\n', x, times2(x));
The result of multiplying 2.00 by 2 is equal to 4.00!

Test Suite

try
% Test 1
assert(isequal(times2(1),2));
% Test 2

% Test 3

% Test 4

catch ME
fprintf('%s\n', ME.message);
end

Solution

function y = times2(x)
 
y = 2 * x;
 
end
